# gonzalez-maria

Proyecto: Registro de Estudiantes

Contenido:
- `index.html` — aplicación en un solo archivo que implementa:
  - Registro de estudiantes.
  - Validaciones: campos obligatorios, email válido, edad entre 18 y 100.
  - Limpia el formulario después de guardar.
  - Eliminar un estudiante (individual).
  - Eliminar todos los estudiantes.
  - Buscar/filtrar registros en tiempo real.
  - Editar registros (botón "Editar").
  - Persistencia local usando `localStorage`.
  - Diseño con la fuente Montserrat y esquema oscuro (coincide visualmente con las imágenes proporcionadas).

Instrucciones:
1. Abrir `index.html` en un navegador moderno.
2. Agregar estudiantes usando el formulario. Después de guardar, los campos se limpian automáticamente.
3. Usar el campo de búsqueda para filtrar por nombre, apellido, email o carrera.
4. Para editar un estudiante, presionar "Editar", modificar los campos y luego "Guardar cambios".
5. Para eliminar todos los estudiantes, usar "Borrar todo".

Nombre de carpeta requerido por la entrega: `primerapellido-primernombre` — en este caso `gonzalez-maria`.

Nota:
- El proyecto usa `localStorage` para persistencia en el navegador; los datos no se envían a servidores.
- Si necesitas el proyecto separado en archivos (HTML/CSS/JS) o un `.zip` con instrucciones adicionales, avísame.
